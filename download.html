User registration

Since EU brain imaging data are considered personal, contracts (i.e., data-use agreements) must be set up between authors and users, which in turn implies that users must be identifiable. Click here to register.

Data Access

Each dataset comes with a Data User Agreement that sets the licensing terms between authors and users. At the moment, you can request data access via the download button from the PublicnEUro catalogue.

Once you have registered, requested a dataset, and signed the DUA, you can download the dataset at any time either using the web browser or via the command line using a link you receive by email. LINKS ARE ACTIVE FOR 72 HOURS ONLY, but you can request a new one when needed:

bash >> wget -qO- https://datacatalog.publicneuro.eu/api/cli.sh

This will prompt you to enter your email, password, the PublicnEUro (PN) ID, and the folder that you want to download (use / for the entire dataset).
