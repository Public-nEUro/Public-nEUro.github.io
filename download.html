User registration

Since EU brain imaging data are seen as personal, one musts set contracts (i.e. data user agreeements) between authors and users, which in turns imply users must be identifiable. To register, please fill out this form with all your information and email it to the Public-nEUro team

Data Access

Each data set comes with a Data User Agreement which sets the licensing terms between authors and users. While we will implement a system to manage these agreements, we are currently in alpha testing phase meaning we are manually checking those signing and providing access to data. Please check our catalogue for which dataset you want to access and email its' ID to the Public-nEUro team.
